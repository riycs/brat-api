# BRAT API USING PLAYWRIGHT

## HOW TO INSTALL

```bash
git clone https://github.com/zennn08/brat-api
cd brat-api
npx playwright install
npm install
node app.js
```

## HOW TO ACCESS API

```
http://localhost:3000/?text=Hi!
```
